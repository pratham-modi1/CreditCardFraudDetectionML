🧠 Key Insight

Overfitting is not controlled by depth alone — it is strongly influenced by model variance.

📌 Decision Tree Behavior
A Decision Tree is a single model
It learns from the entire dataset
It creates hard decision boundaries
It is highly sensitive to:
Noise
Class imbalance
Rare patterns
📉 Result

Even with limited depth, the tree:

Overreacted to minority class patterns
Produced many false positives
Failed to generalize well
3. Introduction of Random Forest

To address the instability of a single tree, a Random Forest model was introduced.

🔧 Key Mechanisms of Random Forest
Bootstrap Sampling
Each tree is trained on a different subset of data
Feature Randomness
Each split considers a subset of features
Ensemble Averaging
Final prediction is based on aggregation of multiple trees
🎯 Effect

Random Forest reduces variance and stabilizes predictions

4. Why Random Forest Works Better Even with Greater Depth

Although each tree in the Random Forest may be deeper (max_depth = 8), the model:

Does not rely on a single tree
Averages multiple decision boundaries
Cancels out noise and overfitting tendencies
🧠 Core Insight
Model	Behavior
Decision Tree	High variance (unstable)
Random Forest	Low variance (stable via averaging)
📌 Conclusion

Depth controls model complexity, but ensemble methods control variance.

Thus, Random Forest can safely use deeper trees while still generalizing well.
